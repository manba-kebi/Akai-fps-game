## 游戏简介

还在为游戏中被外挂玩家虐得死去活来而愤怒吗？还在为举报外挂却杳无音信而无奈吗？

## 游戏玩法

### 基础操作

| 按键 | 功能 |
|------|------|
| W/A/S/D | 前后左右移动 |
| 鼠标移动 | 瞄准 |
| 鼠标左键 | 射击 |
| R | 换弹 |
| 空格 | 跳跃 |
| Shift | 冲刺 |
| 1 | 切换步枪 |
| 2 | 切换手枪 |
| 3 | 切换近战武器 |
| 5 | 打开/关闭作弊菜单 |
| ESC | 暂停游戏 |

### 武器系统

- **M7 Rifle（步枪）**：主武器，高射速，适合中远距离作战
- **Pistol（手枪）**：副武器，作为备用火力
- **Knife（近战武器）**：近距离高伤害，刀人必备

### 回合系统

游戏共5个回合，每回合敌人数量递增，难度逐步提升：
- 第1回合：5个敌人
- 第2回合：10个敌人
- 第3回合：15个敌人
- 第4回合：20个敌人
- 第5回合：25个敌人

### 连杀系统

在5秒内连续击杀敌人会触发连杀音效：
- **单杀**：普通击杀音效
- **爆头击杀**：特殊的爆头击杀音效
- **双杀**：随机播放两个双杀音效之一
- **三杀**：随机播放两个三杀音效之一
- **四杀 / 五杀 / 六杀**
- **七杀及以上**：随机播放两个七杀音效之一

## 特色功能：官方外挂系统 🎮

**重点来了！** 按 `5` 键打开作弊菜单，体验从未有过的"爽感"：

### 可用作弊功能

| 功能 | 说明 |
|------|------|
| 🔫 **锁头（Aimbot）** | 自动瞄准最近的敌人，指哪打哪？不，是打哪中哪！ |
| 🛡️ **无敌（God Mode）** | 敌人打你？不存在的。血条？那只是摆设。 |
| ♾️ **无限弹药** | 换弹？这辈子都不可能换弹的！ |
| 🚀 **飞行模式** | 谁说FPS只能在地上跑？上天！ |

### 使用提示

- 作弊功能可以叠加使用，创造你的专属"神仙模式"
- 单杀时爆头会播放专属爆头音效
- 连杀时无论是否爆头都会触发连杀音效

## 游戏界面

- **左上角**：总击杀数、命中率统计
- **左下角**：弹药数量显示
- **右上角**：小地图（显示敌人位置）
- **屏幕中央**：准星
- **击杀时**：连杀特效、爆头提示
